# E-Commerce Sales Analytics Dashboard

A data analytics project focused on extracting business insights from 500,000+ e-commerce transactions using Python, SQL, Excel, and Power BI.

## Project Overview

This project analyzes large-scale e-commerce sales data to identify revenue trends, customer purchasing behavior, and product performance. The analysis combines data cleaning, transformation, SQL-based querying, exploratory data analysis, and interactive dashboard visualization.

## Key Features

* Processed and analyzed 500,000+ e-commerce transaction records using Python (Pandas) and SQL.
* Performed data cleaning, preprocessing, and exploratory data analysis (EDA).
* Identified sales trends, seasonal patterns, and top-performing products.
* Conducted customer segmentation and purchasing behavior analysis.
* Built an interactive Power BI dashboard for business reporting and decision-making.
* Visualized monthly revenue, regional sales performance, and top customers.
* Discovered that the top 10% of customers generated approximately 35% of total revenue, supporting customer retention and loyalty strategies.

## Technologies Used

* Python (Pandas, NumPy, Matplotlib)
* SQL
* Microsoft Excel
* Power BI

## Dashboard Insights

* Monthly Revenue Analysis
* Top Customers by Revenue
* Regional Sales Performance
* Product Category Performance
* Customer Contribution Analysis
* Sales Trend Forecasting

## Business Impact

The project provides actionable insights that help businesses optimize marketing campaigns, improve customer retention, identify high-value customers, and increase overall revenue performance through data-driven decision-making.
